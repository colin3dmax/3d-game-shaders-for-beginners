3D游戏着色器入门

边缘光（Rim Lighting）

灵感来源于fresnel因子， 边缘光着重突出物体的边缘或轮廓。 结合卡通渲染和描边， 它能完美强化卡通效果。 你也可以用它来高亮游戏中的物体， 帮助玩家更容易导航和完成任务。

  // ...

  vec3 eye = normalize(-vertexPosition.xyz);

  // ...

与fresnel因子一样， 你需要计算视线向量（eye vector）。 如果顶点位置是视图空间的，那么视线向量就是顶点位置的反向。

  // ...

  float rimLightIntensity = dot(eye, normal);
        rimLightIntensity = 1.0 - rimLightIntensity;
        rimLightIntensity = max(0.0, rimLightIntensity);

  // ...

边缘光强度范围是0到1。 当视线向量和法线方向相同时，边缘光强度为0。 两者方向逐渐不同，边缘光强度增加， 当视线与法线正交时，强度达到1。

  // ...

  rimLightIntensity = pow(rimLightIntensity, rimLightPower);

  // ...

你可以用幂函数控制边缘光的衰减曲线。

  // ...

  rimLightIntensity = smoothstep(0.3, 0.4, rimLightIntensity)

  // ...

也可以用step或smoothstep控制衰减， 搭配卡通渲染时效果更佳。 你将在后续章节学习更多相关函数。

  // ...

  vec4 rimLight   = rimLightIntensity * diffuse;
       rimLight.a = diffuse.a;

  // ...

边缘光的颜色由你决定。 示例代码将漫反射颜色乘以边缘光强度， 这样既突出轮廓，又不会过曝或照亮阴影区域。

  // ...

  vec4 outputColor     = vec4(0.0);
       outputColor.a   = diffuseColor.a;
       outputColor.rgb =
           ambient.rgb
        +  diffuse.rgb
        + specular.rgb
        + rimLight.rgb
        + emission.rgb;

  // ...

计算完边缘光后，将其与环境光、漫反射、高光和自发光叠加。
